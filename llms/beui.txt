# beUI

> Bespoke motion components for React.

## Endpoints

- Registry index (JSON): https://beui.dev/r
- Component detail (JSON): https://beui.dev/r/{slug}
- Raw source (text/plain): https://beui.dev/r/{slug}/raw

## Components

- [Tilt Card](https://beui.dev/r/tilt-card): 3D perspective tilt on hover with cursor-tracked glare.
- [Button](https://beui.dev/r/button): Spring-pressed Button plus StatefulButton (idle → loading → success / error) and MagneticButton.
- [Marquee](https://beui.dev/r/marquee): Infinite horizontal or vertical scroll with pause-on-hover.
- [Tabs](https://beui.dev/r/tabs): Pill, segment or underline tabs with a spring layoutId indicator.
- [Switch](https://beui.dev/r/switch): Toggle with a spring-driven thumb and press feedback.
- [Select](https://beui.dev/r/select): Composable select primitives whose panel bouncily unfolds out of the trigger and separates, plus a Morph variant where the trigger grows into the panel via shared layout.
- [Checkbox](https://beui.dev/r/checkbox): Form choice control with a draw-on checkmark, spring press feedback and indeterminate state support.
- [Radio Group](https://beui.dev/r/radio): Single-select choice control with a gliding layoutId indicator dot and spring press feedback.
- [Bottom Sheet](https://beui.dev/r/bottom-sheet): Vaul-inspired draggable bottom sheet with snap points, inertia and glass surface.
- [Shared Layout Background](https://beui.dev/r/shared-layout-bg): A pill that glides between hovered items via motion's shared layout, with blur enter/exit.
- [Dock](https://beui.dev/r/dock): macOS-style dock with grouped actions and a gliding active pill.
- [Tooltip](https://beui.dev/r/tooltip): Hover or focus tooltip with blur enter/exit and spring spawn.
- [Morphing Modal](https://beui.dev/r/morphing-modal): Family-app-style modal. A single panel that morphs its height as you navigate between inner views, with blur cross-fade on content.
- [Text Animation](https://beui.dev/r/text-animation): Animated text primitives for reveal sequences, shimmer loading states and letter-cascade swaps.
- [Number Animation](https://beui.dev/r/number): Animated number primitives for count-up values and rolling digit tickers.
- [Animated Badge](https://beui.dev/r/animated-badge): Status badge with animated state icons, pulse feedback and compact size variants.
- [Action Swap](https://beui.dev/r/action-swap): CTA button and slot primitives for swapping text and icons with blur motion.
- [Animated Toast Stack](https://beui.dev/r/animated-toast-stack): Stacked toasts with status morphs, swipe dismissal, actions and layout-aware motion.
- [Theme Toggle](https://beui.dev/r/theme-toggle): Theme toggle button with a full-page rectangle clip-path reveal via the View Transition API.
- [Bouncy Accordion](https://beui.dev/r/bouncy-accordion): Single-open accordion with weighted spring layout, icon rows and reduced-motion-safe content reveals.
- [Drawer](https://beui.dev/r/drawer): Side panel that slides in from the left or right with a spring, backdrop blur, body scroll lock and esc-to-close.
- [Scroll Animation](https://beui.dev/r/scroll-animation): Scroll-driven motion: a Lenis smooth-scroll provider and a reading-progress indicator that reads from it.
- [Range Slider](https://beui.dev/r/range-slider): Range slider with tick dots and a bouncy vertical-bar thumb that glides between snapped steps; drag and keyboard control, reduced-motion safe.
- [Multi-chain Swap](https://beui.dev/r/swap): Cross-chain swap widget with chain + token selectors, morphing views, animated flip and quote.
- [Dynamic Island](https://beui.dev/r/dynamic-island): iOS-style island pill that morphs between live activity views with bouncy shell resize and blur crossfades.
- [Command Palette](https://beui.dev/r/command-palette): ⌘K palette with fuzzy filter, spring-animated active row and glass surface.
- [Expandable Action Bar](https://beui.dev/r/expandable-action-bar): Compact icon actions that expand into labeled controls on hover or focus with shared layout motion.
- [Overflow Actions](https://beui.dev/r/overflow-actions): Connected pill rail for primary actions that springs open to reveal extra controls.
- [Expandable Tabs](https://beui.dev/r/expandable-tabs): Icon tab bar where the active tab expands to a labelled pill, with a panel above that morphs height and slides content direction-aware on switch.
- [Swipeable List](https://beui.dev/r/swipeable-list): Mobile-style list rows that swipe left or right to reveal contextual action buttons.
- [File Upload](https://beui.dev/r/file-upload): Drag-and-drop upload queue with progress rows, retry/remove actions and reduced-motion-safe state changes.
- [Prediction Market](https://beui.dev/r/prediction-market): Prediction market trade ticket with buy/sell modes, outcome prices, rolling amount entry, quick add chips and trade states.
- [OTP Input](https://beui.dev/r/otp-input): One-time-code input with a gliding focus ring, digits that roll in per slot, error shake and a success check draw.
- [Bloom Menu](https://beui.dev/r/bloom-menu): A button that morphs open into a menu and blooms iris-out from the center, the grid revealing in every direction with radially staggered items.
- [404 / Not Found](https://beui.dev/r/not-found): Animated 404 pages in five styles: glitch scramble, magnetic digits, cursor spotlight, a fanning card stack and a typed terminal.

## Usage for agents

1. Fetch the index to discover components.
2. Fetch a component detail JSON for files, deps and source.
3. Drop files into the user's project at the listed paths.
4. Install external `dependencies` listed on the entry.
5. Internal helpers (e.g. `@/lib/utils`) ship in the `files` array as `type: util`.
